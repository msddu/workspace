index.html 파일은 메인 페이지 역할의 html 파일 입니다

(확인은 web.xml)

서버 제출 테스트1

        -이름과 나이를 입력 받아 서버로제출

        *준비물(선지식)*
        1) input태그 : 값을 입력하는 양식을 제공
            -name속성 : 서버로 제출 시 key 역할을 하는 속성 

            -input에 입력된 값 또는 value 속성의 값 : value역할

        2) form태그 : 내부에 작성된 input값을 지정된 경로로 제출
            -지정된 경로로 제출 == 클라이언트가 서버에 요청 

            -action속성 : 제출할 경로(주소) 
                        (주소 == 무슨 요청인지 지정)

            -method속성 : 제출 방법(데이터 전달 방식)
    
            -action = "/example1"
                ->최상위 주소 /example1 주소로 요청을 보내겠다(제출하겠다)

            -method="GET
                ->제출되는 데어터를 주소에 담아서 제출하겠다
                ex) 주소 ?name = 홍길동&age=20
                            (쿼리 스트링)
